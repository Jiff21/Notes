# Jeff's Notes

Not for duplication or publication.

## Installation
After downloading be sure to run the following commands.

	sudo npm install
	bower install
	grunt build

## Running Locally
Use:

	grunt serve

## Running Tests
To run all tests:

	grunt test

To run Nightwatch tests:

	grunt test:e2e

To run Jasmine Tests (to be written)

	grunt test:unit
