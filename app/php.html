FPO

PHP is a server side scripting language which means it needs to be run on a server. PHP originally stould for Personal Home Page Tools, then it change to PHP: Hypertext Preprocessor (recursive humor). The "Stack" refers to the combination used the most common are MAMP, LAMP, WAMP and XAMP. MAMP stands for Macintosh, Apache, MySQL and PHP (Windows, Linux, and X is wildcard/all). Apache's nickname in terminal is httpd. I'm using MAMP, but if you need to set up an Apache Server you can get it here. You probably want to check out your notes for Terminal as well.

You open and close PHP with the following tags, <?PHP to open and ?> at the end of the doc to close. If you load a PHP file that says <? php phpinfo() ?> it will load the settings of php on the Apache terminal. PHP ignores whitespace and a semicolon ends a line of code.

Exploring Data Types

Variables need to start with a dollar sign, can contain letters, numbers, underscores or dashes, and are case sensitive. Remember to avoid these reserved words. Period's are used for concatenation. You can make a float into an integer with the round() function. When using Arrays it's important to remember that you make an array with parentheses $var = array(x,"word"), unless you're using the new shorthand $var = [x, "wowrd", y] and you call a variable in the array with brackets $var[0]. You can also add and replace to an array using brackets $var[x] = "word" . PHP has a second type of arrays. named Associative Arrays , that have a value and a key. If you use the sort function on an array it will restructure the order of the array. The array_push() function is a handy way to add one more item to the end of an array. array_search() will search for a value and return its corresponding key.

Booleans return a value of 1 if true and nothing if false. You can always use is_bool() to tell if something is a boolean or not. NULL is the term used for a variable not having a value. You can avoid warnings by setting values to Null. You could also use Empty, but that will neturn any of the following values- "", null, 0. 0/.0, "0", false, and array(). The terms Bool and Null are case incesensitive when you write them out. This makes empty powerful but dangerous. If you were using it for form validation, answering a how many children question would lead the validation to thinking the field was empty when a person has no children.

Type Casting is how you can change a string variable into an integer, float, array, etc. Type Juggling is when php automatically changes a string to a variable or vice versa. Juggling will usually occur when you add an integer to a string (which is lazy but not horrible) or when you change an integer to a string using concatination (which should never be done).

Constants can only be changed with the define("VAR") functions, they are recognizable because they are written in all caps and don't have a dollar sign in front of them. Once you define a constant it can't be changed with functions or even a second time with the define function. Constants only last for the length of the php script.
