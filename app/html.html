Let's talk about CSS (H3 of 6 Header).

This is the Sixth Paragraph. In it we're going to talk about CSS. When writing CSS you can just style by using different element names on the page by using them as a selector. If you want to select a class, you need to put a period in front of the name (e.g. .example:). If you want to use an ID selector, use the Pound symbol in front of the name (e.g. #example:) Some commonly used CSS properties are font-family, font-size, font-weight, border, margin, background-color, , . Setting the overflow property to auto is a neat trick. When you are working with borders always remember TRBL (Top, Right, Bottom, Left). You can add multiple classes to a html element using whitespace. Click on the picture to go back to the top of the document:

Selectors are very important in styling your page. Descendant Selectors just require a space (e.g. h2 p). Child selectors are similar but they don't target the whole page (e.g. h2 > p). Adjacent selectors (e.g. h1 + p, H2 +p,) can be useful if there if you want to select something that follows another thing directly and is part of the same parent element. Attribute Selectors look for things with the thing in brackets (e.g. a[title] or a[title="example"]). When using Attribute selectors you can use a tilde (~) before the equal sign to find anything with it instead of an exact match or you can use a carrot (^) to look for anything that starts with. Or you can use a dollar sign ($) to look for anything that has that end. As you're familiar with you can use asterisk to do a wildcard search (*).
