To add a login banner use the command sudo defaults write /Library/Preferences/com.apple.logi­nwindow LoginwindowText "Use of the system implies consent to user monitoring, any unauthorized use of the system will be prosecuted." To remove the banner use "sudo defaults delete /Library/Preferences/com.apple.logi­nwindow LoginwindowText".
