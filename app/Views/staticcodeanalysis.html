QARK

Description

QARK (Quick Android Review Kit) is a tool that does static code analysis on Android App to locate possible security issues, it can examine source code or an APK. It then provides an HTML report of known vulnerabilities and can compile a QARK app that will allow you to test possible vulnerabilities on a device or emulator.

Install & Run

All you really need to do is clone QARK

git clone git clone https://github.com/linkedin/qark.git

cd into the folder, and run with the follind command.

python qark.py

You can simply follow the prompts to get a report and run tests against evil twin of the app.

Retire.js

Installation

Retire.JS is the javascript equivalent to Dependency Check, it does static code analysis and lists know vulnerabilities with components you are using. As long as you have NPM it's super simple just run the following command.

sudo npm install -g retire

Instructions

If you want to see vulnerabilities in the Terminal, simply run:

--retire --path ~/Path/To/Project/Root/

I prefer to read them in a text file, so I can separate issues and mark things off as I go. This command will write a file in your Documents folder with whatever name you choose, output format can be .txt or .xml.

retire --path ~/Path/To/Project/Root/To/Test/ --outputpath ~/Documents/AnyFileName.txt
