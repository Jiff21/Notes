Basics

cd means change directory.

cd .. goes back a folder.

ls shows the files in that directory.

open . open the folder you are currently in.

cat filename.txt to read a file.

VIM

To edit a file: vim filename.txt.

Press i button to enter insert mode.

ctrl + c to exit insert mode.

To write quit hit : then type wq.

Synced Bash Profile

All your bash profile settings can be setup by creating a .bash_profile file in the home directory and adding this line:

source /Applications/MAMP/htdocs/No­tes/app/examples/.bash_profile

Bash Shortcuts

Terminal shortcuts are store in the .bash_profile in the User folder on mac. By default this folder is hidden. You can make a shortcut a command or location with alias newShortcutName="any terminal command".

Change Command Line

This is your command line preference for .bash_profile

if [ -f ~/.git-prompt.sh ]; then
    source ~/.git-prompt.sh
    export PS1='[\u \W$(__git_ps1 " (%s)")]\$ '
fi

Git Auto-Complete

The following addition to .bash_profile will turn on git auto-complete:

if [ -f ~/.git-completion.bash ]; then
    source ~/.git-completion.bash
fi

Syncing Submlime Settings

Currently you can sync your sublime 3 settings with the following commands. Be sure to quit sublime before doing remove.

cd ~/Library/Application\ Support/Sublime\ Text\ 3/Packages/
rm -r User
ln -s /Users/$USER/Documents/Code/Not­es/app/examples/sublime3/User

Sublime-Linter setups

jsHint

It must be installed Gloobally:

sudo npm install -g jshint.

If you use a variable that is global and you don't want it to warn you.

/* global variableName */

If you use are exporting a variable or function and you don't want it to warn you.

/* exported variableName, functionName */
scss & sass linter

Install ruby if not already installed.

sudo gem install scss-lint
php linter

PHP must be installed. Shouldn't be a problem.
