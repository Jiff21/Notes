Starting a Project

Locally use:
git init
touch README.md
git add *.c
git add README.md
git commit -m "This is the 1st commit"

Pull down a project that has already been started:
git clone github/url

Stash

Save current changes to Stash
git stash -m "Message about stash"

List stash changes
git stash list

Get a stashed change back to working directory:
git stash pop HashFragNum

Clear all stashed changes:
git stash -m "Message about stash"
