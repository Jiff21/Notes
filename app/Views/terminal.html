Basics

cd means change directory.

cd .. goes back a folder.

open . open the folder you are currently in.

cat filename.txt to read a file.

VIM

To edit a file: vim filename.txt.

Press i button to enter insert mode.

ctrl + c to exit insert mode.

: then type wq to write quit.

Bash Shortcuts

Terminal shortcuts are store in the .bash_profile in the User folder on mac. By default this folder is hidden. You can make a shortcut a command or location with alias newShortcutName="any terminal command".

Change Command Line

This is your command line preference for .bash_profile

if [ -f ~/.git-prompt.sh ]; then
    source ~/.git-prompt.sh
    export PS1='[\u \W$(__git_ps1 " (%s)")]\$ '
fi

Git Auto-Complete

The following addition to .bash_profile will turn on git auto-complete:

if [ -f ~/.git-completion.bash ]; then
    source ~/.git-completion.bash
fi
