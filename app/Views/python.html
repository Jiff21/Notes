Sets & Frozen Sets

Being an unordered collectio of unique elements, sets do not record element position or order of insertion. Accordingly, sets do not support indexing, slicing, or other sequence-like behavior. Sets are mutable and frozensets are not.

Boolean Operators

Order of precedence is not, and, or.

  • not not True is false
  • and True and False is False
  • or or | True or False is True

Combining vars with strings

While print "Hey " + you + " there." would need spaces inserted. If you need to do it without spaces, you could do print "He's got %s eyes and %s hair." % (my_eyes, my_hair).
