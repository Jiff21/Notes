Built In Functions

Full list here. Ones you are used to here.

  • str(): to String.
  • len(): length of many things
  • bool(2 > 3): Returns True/False Boolean
  • callable(func): Lets you know if an object is callable
  • classmethod(): Returns class method of a function.
  • list(enumerate(seasons)): Returns a tuple containing count and value like from an array.
  • getattr(): getattr(x, 'foobar') is equivalent to x.foobar.
  • globals(): Returns all gloabls
  • locals(): Returns a dict representing the current local symbol table.
  • map(function, iterable, ...): Apply function to every item of iterable and return a list of the results.
  • open(name[, mode[, buffering]]): See here.
