Built In Functions

Full list here. Quick and easy here.

  • print("Hello world"): Or print "Hello World"
  • str(): to String.
  • len(): length of many things
  • bool(2 > 3): Returns True/False Boolean
  • callable(func): Lets you know if an object is callable
  • classmethod(): Returns class method of a function.
  • list(enumerate(seasons)): Returns a tuple containing count and value like from an array.
  • getattr(): getattr(x, 'foobar') is equivalent to x.foobar.
  • setattr(object, name, value): setattr(x, 'foobar', 123) is equivalent to x.foobar = 123.
  • globals(): Returns all gloabls
  • locals(): Returns a dict representing the current local symbol table.
  • map(function, iterable, ...): Apply function to every item of iterable and return a list of the results.
  • range(0, 30, 5): 0, 5, 10,...,30.
  • list(reversed(counting)): [3,2,1]
  • round(3.1234,2): 3.12, 5 rounds up
  • sorted(iterable[, cmp[, key[, reverse]]]): Sort an object, details here. See here.
  • a = raw_input('Prompt text here'): Prompt user for input See here.
  • help()Command line help module, like pydoc but that is outside running python

Formatters

  • %s = variable for string replacements
  • %r = raw data for debugging, like strings display quotes around them
  • %d = for numbers not string
  • %% = Escape Percentage Symbol

Escape Sequences

  • \n = New line in a string of text.
  • \'= Esccape a single quote.
  • \" = escape a double quote
  • \t = tab
  • \\ = backslash
  • \N{name} = Character name in Unicode
  • \a = ASCII Bel (Beep)
  • \b = backspace
  • \xhh = Charcter with hex value of
  • u'\U0001F47E' = Unicode String

File Functions

  • close: Closes the file. Like File->Save.. in your editor.
  • open(name[, mode[, buffering]]): Reads the contents of the file. You can assign the result to a variable. Defaults to r mode, need to pass 'w' if you want to write See here.
  • readline: Reads just one line of a text file.
  • truncate: Empties the file. Watch out if you care about the file.
  • write('stuff'):Reads just one line of a text file.
  • var.read():After you open something, if you need the string rather than the object

There is a difference between commas and plus's when combining strings. For example print "Hey" , var , "there." would print "Hey you there." (if var = you). While print "Hey " + you + " there." would need spaces inserted. If you need to do it without spaces, you could do print "He's got %s eyes and %s hair." % (my_eyes, my_hair). It's bad form to have more than 80 characts on a line but you can two prints with a comma.
