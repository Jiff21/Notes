Syncing Submlime Settings

Currently you can sync your sublime 3 settings with the following commands. Be sure to quit sublime before doing remove.

cd ~/Library/Application\ Support/Sublime\ Text\ 3/Packages/
rm -r User
ln -s /Users/$USER/Documents/Code/Not­es/app/examples/sublime3/User

Sublime-Linter setups

jsHint

It must be installed Globally:

sudo npm install -g jshint

If you use a variable that is global and you don't want it to warn you.

/* global variableName */

If you use are exporting a variable or function and you don't want it to warn you.

/* exported variableName, functionName */
php linter

PHP must be installed. Shouldn't be a problem.

scss & sass linter

Install ruby if not already installed.

sudo gem install scss-lint

Trying to figure out how to turn off indentation with spaces. Might be go /Library/Ruby/Gems/2.0.0/gems/ find sass-linter version go into config folder and edit default.yml file to have the following settings for Indentation character:tab & Indentation:width:1, then restart.

html-tidy

Should be preinstall on OSX but you can update with:

brew install homebrew/dupes/tidy

or get the html 5 version:

brew install homebrew/dupes/tidy --HEAD

Useful Shortcuts

  • Global Find and Replace = cmd+shift+F
  • Command Pallet = cmd+shift+p
  • File Search = cmd+p then filename#term
  • Move up a line = cmd+cntl+↑
  • Indent back = shift+tab

More Tips

You know the command pallet is how you get to package manager, but it can also be used to toggle settings, like word wrap.
